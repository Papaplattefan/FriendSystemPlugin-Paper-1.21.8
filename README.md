# FriendSystem

Ein simples Friendsystem für Paper 1.21.x

## Features
- /friend add, remove
- /friend request, accept, deny
- GUI mit PlayerHeads
- Online/Offline Status
- Persistente Speicherung (YAML)

## Kompatibilität
- Paper 1.21.x

## Commands
/friend add <Spieler>
/friend remove <Spieler>
/friend list
/friend request <Spieler>
/friend accept <Spieler>
/friend deny <Spieler>
